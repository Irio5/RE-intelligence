# RE Intelligence

Dashboard di business intelligence immobiliare per acquirenti casa.

Analisi basata su transazioni reali: prezzi per zona, trend di mercato, stime, confronti e comparabili — con spiegazioni chiare per chi deve comprare.

Fase 1: Milano. Obiettivo: tutta Italia.

## Tech Stack
- Next.js (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- Recharts + Leaflet
- Supabase (PostgreSQL + Auth)
